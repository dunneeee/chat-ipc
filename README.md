# Multi-Process Chat Server using Socket IPC in C

## How to run

1. Run `make` to compile the server and client.

```bash
make
```

2. Run the server in one terminal.

```bash
./server
```

3. Run any number of clients in other terminals.

```bash
./client <input_file> <output_file>
```

4. Type your message in the client terminal and press enter to send it to the server.
